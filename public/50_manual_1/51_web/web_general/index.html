色

Webセーフカラー web216 - Web Safe Colors

基本はダークモードを使用していない状態で確認する。webセーフカラーを使っておけば思ったのと違う色になることはほとんどない。

webセーフカラーに含まれない色を使ったとしてもそれで問題になるほど違う色になることはあまりないのでそこまで意識しすぎる必要もない。使いたい色がwebセーフカラーにないときは自分の使いたい色を優先しても問題ない。
