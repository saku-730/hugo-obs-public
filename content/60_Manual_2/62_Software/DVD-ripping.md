映像を残し続ける。

## 手順

1. Braseroでディスクのコピーからisoファイル(イメージファイル)作成する
2. HandBrakeで1で作成したisoファイルをソースに選択しMPEGでリッピングを行う

## ソフトウェア

### Brasero

DVDリッピングソフト。DVDのリッピングだけでなく書き込みなども可能。

インストール方法↓
```
sudo apt install brasero
```

### HandBrake

DVDをMP4/MKV/WebMに変換する。
汎用性の高さからMP4を使う。

インストール方法↓

```
sudo apt install handbrake
```
