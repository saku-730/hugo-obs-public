## Github Actions

ビルド、テスト、デプロイなどをGithub上で自動化して行うためのもの。ある程度決まりきった処理を特定の時間や特定のタイミングに行うといった処理に最適。

### 書き方

#### 実行タイミング

##### 定期実行

一日一回や一時間おきなど定期的に実行したいときはcronをつかう。

##### 行為をトリガーにする

リポジトリにpushされたら、といったように特定の行動が行われたときに実行したいときの方法。
