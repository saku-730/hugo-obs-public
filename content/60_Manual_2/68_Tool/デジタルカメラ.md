# デジタルカメラ

## 選び方

- センサーサイズはAPS-Cでも全然いい
- レンズはマクロを昆虫用に

## カメラの状態

exiftoolで撮影した画像ファイルから取得する。

```bash
sudo apt update 
sudo apt install libimage-exiftool-perl
```

↑でインストールして

```bash
exiftool [ファイル名]
```

## 設定 D7100

- ISO感度：オート
- フォーカス：基本オート、特定のものにフォーカスしたいときだけ、いじる
- 絞り F値：マニュアル
- シャッタースピード：オート

↑の設定にするため、ISOオートを有効にした上で、Aモードに設定。
