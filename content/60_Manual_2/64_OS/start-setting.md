## Ubuntu

```bash
sudo apt update
sudo apt upgrade -y
sudo apt autoremove -y

sudo apt install stress -y  # s-tuiに必要
sudo apt install s-tui -y # ストレステスト CPU温度をみて冷えているか確認

sudo apt install htop -y #システム監視

sudo apt install build-essential -y #主要ビルドツール

sudo apt install vim -y

sudo apt install cifs-utils -y

sudo apt install dkms -y
```
