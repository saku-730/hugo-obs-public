# Windows

## Windows11

### ハードウェア要件

- CPU 2コア
- メモリ 4GB(推奨8GB以上)
- ストレージ64GB(推奨128GB以上)

### ライセンス

リテールライセンスは移行可能だが、いくつか注意点がある。
バージョン(主にHomeかProか)があっていないと、ライセンスの使い回しができない。isoイメージはバージョンそれぞれ同じものでインストールの過程で選ぶので注意する。持っているライセンスはPro。

移行するためにはMicrosoftアカウントに紐づけておくのが一番簡単。

プロダクトキーがわからなくなってもアカウント紐付けをしておけば大丈夫。

### インストール

isoを公式からダウンロードして書き込むのはLinuxなどと同様であるがなぜかWindowsPCでやらないとうまくいかないことが多い。書き込みは成功するが起動しなかたり起動しても途中で進めなくなったりする。WindowsPCでLiveUSBはつくったほうがいいみたい。
つくづく面倒なOSである。
