# 生殖概論

## 生殖形式

- Parthenogenesis Thelytoky 単為生殖 : 精子を用いず、卵子のみでクローンを作成し、繁殖する
	- Automictic parthenogenesis : 減数分裂は行われるが、染色体数の倍増によって補填される。
	- Apomictic parthenogenesis :  減数分裂がおこらない。
- Gynogenesis 雌性発生 : 精子の受精は起こらないが、精子によって発生が起動する。雌の遺伝子のみ、遺伝される。二倍体の卵子である。 Pseudogamy
- Androgenesis 雄性発生 : 雄の遺伝子のみ遺伝する。卵はミトコンドリア以外は遺伝しない。
- Hybridogenesis : 交雑
- Asexual reproduction : 体細胞から新たな個体が形成される。
- Anisogamy : 有性生殖の中で、配偶子の大きさが付均等なもの。小さいほうが雄、大きいほうが雌と呼ばれる。↔Isogamy
- Apomixis : 無性生殖の一つ。減数分裂が抑制され、有糸分裂による分裂が一度行われる。子孫は母親と遺伝的に同一。
- Arrhenotoky : 未受精卵から雄が作られる形式。膜翅目の半倍数システムなど。
- Automixis : 無生成色の一つ。減数分裂が正常に行われ、4つのハプロイドを生産する。二倍体は2つの前核、または早期開裂核の融合によって作られる。
- Dioecious : 雌雄異株を指し、主に植物に用いる。
- Gonochoric : 雌雄異体を指し、主に動物に用いる。
- Karyogamy : 2つの核の融合。
- Syngamy : 2つの配偶子が融合して接合体を作ること。

### Parthenogenesis

単為生殖

精子を用いない、形式の繁殖方法。ただし、卵細胞からの発生であり、体細胞から発生が起こるAsexual reproduction 無性生殖とは区別される。ここで、精子を用いないとは精子の遺伝情報が子個体に遺伝しないだけでなく、精子が発生に一切必要でないことを指す。これはGynogenesis 雌性発生との区別を目的としている[2]。

単為生殖の中でも、減数分裂を経ているかどうかや、倍数体の数によって更に分けられる。
